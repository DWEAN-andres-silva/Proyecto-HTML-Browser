HTML
HYPER TEXT MARKUP LANGUAGE (LENGUAJE DE MARCAS DE HIPER TEXTO) Proyecto 2 - HTML

HTML es el elemento de construcción más básico para crear y representar visualmente una página web, más no su funcionalidad.

Es clave ya que las siglas M y L (Hyper Text) son las responsables de enlazar páginas o diferentes sitios web.

M (Markup) son los tags que sirven para "anotar" los elementos de la página.

JAVASCRIPT
EL LENGUAJE POR DEFECTO DE SCRIPT PARA LAS PÁGINAS WEB Proyecto 2 - Javascript

JAVASCRIPT, a menudo llamado js, es un lenguaje interpretado multi paradigma.

Esta desarrollado bajo el estándar ECMAScript.
Su última versión es la 5.1.

Javascript es el lenguaje que está encargado de la funcionalidad de los elementos que componen la página web.

CSS
HOJAS DE ESTILO EN CASCADA Proyecto 2 - CSS

Las CSS describen la presentación de los documentos HTML.

Su última versión es la 3, siendo CSS3 un lenguaje base que está por convertirse en estándar en la Open Web.

Existen diferentes frameworks que usan CSS3 como base, entre muchos se encuentra SASS, que funcionan como un pseudolenguaje que añade funcionalidad a elementos o estilos de las páginas web, tales como animaciones.
